# Медичний лендинг — E-sheba

Навчальний проєкт із фронтенд-веб-розробки, виконаний на основі макету з Figma Community. Мета проєкту — відпрацювання навичок семантичної верстки, адаптивного дизайну (Responsive Web Design) та базового інтерактиву за допомогою JavaScript.

**Посилання на макет:** [Figma Community — Medical Landing Page Design](https://www.figma.com/community/file/1170008021377081653)

---

## 🚀 Технологічний стек

* **HTML5** — семантична та доступна розмітка.
* **CSS3** — методологія БЕМ (Блок, Елемент, Модифікатор), Flexbox, CSS Grid для побудови сіток.
* **JavaScript (Vanilla JS)** — реалізація інтерактивних елементів.
* **SVG** — використання векторної графіки для іконок, кастомних підкреслень та фонових фігур.
